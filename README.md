# 🎓 Student Management System & Graphical Dashboards (Tkinter Desktop App)

A modern, light-weight, and highly intuitive Desktop Application designed to streamline academic administration and student data tracking. Built using **Python's Tkinter Framework**, this system provides separate, secure graphical interfaces (GUIs) for both Administrators and Students to handle full database CRUD operations efficiently.

---

## 🚀 Core Modules & Features

*   **Interactive Gateway (`Home_Page.py`):** A clean landing GUI screen that lets users choose between Admin and Student portals.
*   **Dual Authentication Portals:** Secure desktop login windows managed via `Admin_login.py` and `Student_login.py`.
*   **Admin Dashboard (`Admin_Dashboard.py`):** A centralized control panel featuring modern navigation widgets to oversee entire campus metrics.
*   **Complete Student CRUD Engine:** Robust graphical forms handling back-end student database records seamlessly:
    *   `Add_student.py`: GUI form to onboard new students with automatic validation.
    *   `Update_student.py`: Interactive window to search and modify existing records.
    *   `Delete_student.py`: Secured tool to remove records safely from the database.
*   **Student Dashboard (`student_Dashboard.py`):** A personalized desktop layout allowing students to view their profile details, attendance, and academic grades.

---

## 🛠️ Tech Stack Used

*   **Language:** Python 3.x
*   **GUI Framework:** Tkinter (Python standard GUI library)
*   **Image Processing:** Pillow (PIL) — for rendering dynamic dashboard icons and branding assets.
*   **Database:** SQL / SQLite (Structured backend to store credentials and student rosters)
